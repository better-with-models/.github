# Better with Models

Open-source infrastructure for AI-assisted development, education, and operations.

This is where the more ambitious projects live — tools and systems that explore how to build reliable, productive workflows around large language models. From intelligent inference routing to Claude plugins, containerized compute environments, and documentation-driven governance, the work here sits at the intersection of software engineering and applied AI.

Each repository is self-contained with its own documentation and tooling. Browse the repos to see what's in progress.
